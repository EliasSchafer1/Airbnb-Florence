# Airbnb Florence

Data Science project exploring the factors that influence Airbnb prices in Florence using statistical analysis and linear regression

## Research Question

Which factors explain the price differences of Airbnb listings in Florence?

## Requirements

- Python 3.11+
- Poetry

## Setup

### 1. Clone repository:
```
git clone https://github.com/EliasSchafer1/Airbnb-Florence.git
cd Airbnb-Florence
```

### 2. Set up the python version (e.g. via pyenv): 
```
pyenv install 3.11.8
pyenv local 3.11.8
```

### 3. Install dependencies:
```
poetry install
```

### 4. Run notebooks:

Run the notebooks in order: `01_preprocessing.ipynb` → `02_exploration.ipynb` → `03_analysis.ipynb`

## Data

The dataset contains Airbnb listings in Florence, scraped on September 22, 2025. It was obtained from Inside Airbnb (http://insideairbnb.com) in April 2026.
